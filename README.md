# Python 24点纸牌游戏

一个基于Python和Pygame开发的24点纸牌游戏，具有图形化界面、音效和完整的游戏功能。

## 游戏简介

24点游戏是一种经典的数学益智游戏，玩家需要利用给定的4个数字，通过加、减、乘、除四则运算，使计算结果等于24。本游戏提供了友好的图形界面和丰富的交互体验。

## 功能特点

- 🎮 **图形化界面**：使用Pygame开发的精美游戏界面
- 🔊 **音效系统**：包含背景音乐、胜利音效、失败音效和警告音效
- 🎯 **智能生成**：自动生成有解的24点题目
- 💡 **答案提示**：提供多种解题方案供参考
- 🔄 **操作便捷**：支持重置、下一题、查看答案等功能
- 🎨 **美观设计**：精心设计的卡片样式和色彩搭配

## 安装指南

### 环境要求

- Python 3.6+
- Pygame 库

### 安装步骤

1. 克隆或下载项目到本地
2. 安装依赖库：
   ```bash
   pip install -r requirements.txt
   ```

3. 运行游戏：
   ```bash
   python Game.py
   ```

## 游戏规则

1. 游戏会随机生成4个数字（1-10）
2. 玩家需要选择两个数字和一个运算符进行计算
3. 计算结果会替换原来的两个数字
4. 重复步骤2-3，直到只剩下一个数字
5. 如果最终数字等于24，游戏胜利；否则游戏失败

## 操作方法

- **数字卡片**：点击选择两个数字进行计算（最多选择2个）
- **运算符卡片**：点击选择一个运算符（+、-、×、÷）
- **RESET按钮**：重置当前题目到初始状态
- **ANSWERS按钮**：查看当前题目的解题答案
- **NEXT按钮**：进入下一道题目

## 技术实现

### 核心算法

- **题目生成**：随机生成4个1-10的数字，确保题目有解
- **答案验证**：使用递归算法验证所有可能的运算组合
- **分数计算**：支持分数运算，避免浮点数精度问题

### 界面设计

- 使用Pygame的Sprite系统管理游戏元素
- 响应式卡片设计，支持鼠标悬停效果
- 精心设计的色彩方案和布局
